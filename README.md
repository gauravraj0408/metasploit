# Metasploit
Metasploit Installation Script


--------------- Installation Steps --------------

## Installation

clone this repository
```
git clone https://github.com/hackers-brain/metasploit.git
```
change the directory to metasploit
```
cd metasploit
```
Give executable permission to the file
```
chmod +x metasploit.sh
```
now run the script
```
./metasploit.sh
```
